Chatbot for Ports - A Demo

This project was created at 2018, while I was a student, for the "Advanced User Interfaces" course.

The aim of this project was to create a chatbot for the port of Ermoupolis, Syros, which would show users the port schedules they wanted to see, based on their given criteria. This chatbot would utilize text-to-speech, in order to be usable by people with sight problems.

Currently, this chatbot demo is only available in English and uses test data.

It was built using Unity Engine and the AIML language.
